## License Details

**Server-bound:**
Your license is tied to one server IP address. If you migrate to a new server, you can update the licensed IP through your [UnderHost client area](https://customerpanel.ca).

**No per-account fees:**
Once the server is licensed, you can provision Redis instances for as many cPanel accounts as the server can support. There is no per-account or per-instance charge.

**No expiry:**
Your license does not expire. You keep the software and all updates indefinitely.

**No recurring fees (currently):**
The current model is a single one-time payment. There are no monthly or annual fees.
