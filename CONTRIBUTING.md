# Contributing to cPanel Redis Manager

Thank you for your interest in contributing to the cPanel Redis Manager project.

---

## Scope of This Repository

This repository contains **documentation only**.

cPanel Redis Manager is a proprietary commercial product developed and maintained by [UnderHost](https://underhost.com). The plugin source code is not publicly distributed, and code contributions to the plugin itself are outside the scope of this repository.

---

## How You Can Contribute

### Documentation Fixes and Improvements

You are welcome to contribute:

- Typo fixes
- Clarifications
- Broken link fixes
- Documentation consistency improvements
- Compatibility or wording updates that match officially published product behavior

### Suggested Contribution Flow

1. Fork this repository
2. Create a branch, for example:
   ```bash
   git checkout -b docs/fix-description
   ```
3. Make your changes
4. Commit with a clear message, for example:
   ```bash
   git commit -m "docs: clarify installation overview"
   ```
5. Open a pull request with a short explanation of what changed and why

---

## Documentation Standards

Documentation contributions should:

- Stay accurate and consistent with the official website at [cpanelredismanager.com](https://cpanelredismanager.com)
- Match the current public roadmap and release status
- Keep the tone clear, technical, and concise
- Avoid invented features, pricing, dates, or claims not officially announced

Please do **not** include:

- Private installer URLs or download endpoints
- Proprietary source code
- Reverse-engineered implementation details
- Claims about unsupported platforms or unreleased functionality unless already documented in the roadmap

---

## Bug Reports and Feature Requests

If you encounter a problem with the installed plugin or want to suggest a feature:

1. Check the [existing issues](https://github.com/UnderHost/cPanelRedisManager/issues)
2. Open a new issue with as much context as possible
3. For plugin issues, include:
   - operating system
   - cPanel version
   - Redis version
   - plugin version
4. For feature requests, describe the real use case

---

## Security Vulnerabilities

**Do not report security vulnerabilities through GitHub Issues.**

Please follow the responsible disclosure process described in [SECURITY.md](SECURITY.md).

---

## Out of Scope

The following are outside the scope of this repository:

- Pull requests that add or modify proprietary plugin source code
- Requests to publish the plugin as open source
- Attempts to document private installation packages or internal distribution methods

---

## Code of Conduct

Be respectful, constructive, and specific.

We are a small team and genuinely appreciate thoughtful feedback that improves the documentation and the product experience.
